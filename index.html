Pre Inconsistency Detection Steps:

  • Create New Folder In One Drive And Name It Data Lake
  • Inside The Folder Create A Excel Workbook And Name It Data Cleaning
  • Copy And Paste Table Into Sheet1
  • Rename Sheet1 As MessyData
  • Create New Sheet And Rename CleanedData
  • In CleanedData Copy The Messy Data From MessyData And Paste As Values
  • In CleanedData, Click On Insert From The Ribbon
  • Click Anywhere Inside The Data Range
  • Click Table
  • Click OK
  • Click Table Design From Ribbon If Not Already Selected
  • Change Table Name To data_set

  • Click On The Insert Drop Down
  • Click Insert Table Columns To The Left
  • Rename Column ID
  • Highlight All Available Table Cells Inside The New ID Column
  • Click Format â†’ Select Number â†’ Reduce Decimal Places To 0

Second Inconsistency Detected:

Inconsistent Casing In Type & Item Columns

Solution:

  • The Data As Noticeable Stretches From Column A To Column J
  • Click In Column L Cell 2 (One Column Away From The Table So New Table Column Is Not Created, This Is A Helper Column)

  • This Will Capitalise Each First Word In The Cell
  • Hover Over The Bottom Right Of The Cell
  • Click And Drag To The Next Cell And Any Subsequent Cells Until All The Data Of The Column Has Been Effected
  • Copy All The Cleaned Cells
  • Paste As Values Into The Table Column For Type
  • Delete The Helper Column And Clear Contents
  • Click In Column M Cell 2 (As This Is A Helper Column)

  • This Will Capitalise Each First Word In The Cell

  • Click And Drag To The Next Cell And Any Subsequent Cells Until All The Data Of The Column Have Been Effected

Third Inconsistency Detected:

Use Of Currency Symbols In Unit Price & Total Columns

Solution:

  • The Table As Noticeable Has A Metadata Column Named Currency To Seperate Meaning From Value
  • The Unit Price & Total Columns Need To Be Formatted As Number With Decimals For Calculation Integrity
  • Select All The Table Cells In The Unit Price Column
  • Right Click And Click On Format Cell
  • Select Number, Change To 2 Decimal Places If Not Selected As Default, And Click Ok
  • As Noticeable All The Currency Symbols Have Been Safely Removed
  • Highlight The Table Cells In The Total Column, Right Click, Click On Format Cell
  • Select Number, Change To 2 Decimal Places If Not Selected As Default, And Click Ok
  • As Noticeable All The Currency Symbols Have Been Safely Removed
  • Click In Column L Cell 2
  • In The Formula Bar Type =IF(OR(ISNUMBER(F2),F2=""),F2,VALUE(SUBSTITUTE(SUBSTITUTE(F2,"USD",""),"GBP",""))) And Hit Enter
  • "If the cell already is a number or is blank, leave it alone. Otherwise, strip out USD/GBP text and convert the result into a proper number."
  • Hover Over The Bottom Right Of The Cell
  • Click And Drag To The Cell Underneath And Keep Doing This Until All The Columns Have Been Effected
  • Copy All The Cleaned Cells
  • Paste As Values Into The Table Column For Unit Price
  • Delete The Helper Column And Clear Contents
