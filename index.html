üîé The Agent‚Äôs Actions

  1. Load the datasets

    Reads the private CSV (with sensitive fields like name, email, phone, DOB, address).

    Reads the sales CSV (with public fields like product_code, category, notes, etc.).

    Stores them in memory as Pandas DataFrames.

  2. Clean sensitive fields (local rules)

    Phone numbers: Normalize to US format (+1 1234567890).

    Emails: Lowercase and trim whitespace.

    DOB: Convert to ISO format (YYYY-MM-DD).

    Addresses: Collapse extra spaces.

    Logs every change with before/after values, rule applied, and timestamp.

  3. Clean public fields (AI‚Äëassisted rules)

    Sends a sample of the sales data to the AI model with rules like:

    • Product codes should be uppercase and padded (BAT-001).
    • Categories should be canonical (Bats, Balls, Helmets, etc.).
    • Notes should be trimmed.

    AI suggests transformations.

    Applies them row by row.

    Logs every change with before/after values, rule type, and timestamp.

  4. Merge datasets

    Joins private and sales tables on customer_id.

    Creates a unified dataset with both sensitive and public fields.

  5. Redact sensitive fields for audit

    Replaces sensitive values with placeholders ({{EMAIL}}, {{PHONE}}) before sending to AI for reporting.

    Ensures sensitive data is never exposed outside the app.

  6. Generate audit report

    Asks AI to summarize:

    • Inconsistencies found.
    • Fixes applied.
    • Example rows (with placeholders).
