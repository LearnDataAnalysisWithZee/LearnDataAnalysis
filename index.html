Data Analytics - Evolutionary Ladder

  • Essential Theory: GDPR-Compliant Sentence To Schema Operational Dataset Designer
  • Data Collection: Dataset Entry Form, Capture Input
  • Data Preparation: Inconsistency Detection -> Standardise -> Outlier Detection -> Flag For Review -> Correct
  • Data Analysis: Strategic Questions, Dashboard
  • AI: Analysed Data -> Connected To An LLM -> Using Prompts, To Generate Reports And Increase Insight
