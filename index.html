Chapter 1

Foundations

The mindset and workflow behind effective analysis.

What data analysis is

Data analysis is the process of turning raw data into useful insight for decisions. It involves asking good questions, cleaning data, exploring patterns, and communicating results.

A simple workflow

  1. Define the question
  2. Collect & understand the data
  3. Clean & prepare
  4. Explore (EDA)
  5. Model or summarise
  6. Communicate results
# Example (pseudo/Python-ish)
question = "What drives customer churn?"
data = load("customers.csv")
clean = clean_data(data)
insights = explore(clean)
report(insights)

Exercises

  • Write 3 business questions you’d like to answer with data.
  • Find one dataset and identify 5 columns you don’t understand (yet).
  • Describe how you’d measure “success” for your question.

Chapter 2

Cleaning & Preparing Data

Make messy data usable and trustworthy.

Common data issues

  • Missing values
  • Duplicates
  • Inconsistent categories (e.g., “UK”, “U.K.”, “United Kingdom”)
  • Bad types (numbers stored as text)
  • Outliers and impossible values

Cleaning checklist

Structure

  • One row = one observation
  • One column = one variable
  • Consistent column names

Quality

  • Handle missing values intentionally
  • Remove/merge duplicates
  • Validate ranges & types
# Example cleaning snippet (Python)
df.columns = [c.strip().lower().replace(" ", "_") for c in df.columns]
df = df.drop_duplicates()
df["age"] = pd.to_numeric(df["age"], errors="coerce")

Exercises

  • Pick a dataset and list 10 potential cleaning checks.
  • Find duplicates: what counts as “duplicate” in your data?
  • Choose a missing-value strategy for 2 columns and justify it.

Chapter 3

Exploratory Data Analysis (EDA)

Understand distributions, relationships, and patterns.

Summary statistics

Start with counts, means/medians, spreads, and simple group comparisons.

# Example EDA (Python)
df.describe(include="all")
df.groupby("segment")["revenue"].mean().sort_values(ascending=False)

Visual exploration

  • Histograms for distributions
  • Box plots for spread/outliers
  • Scatter plots for relationships
  • Bar charts for comparisons

Rule of thumb

If you can’t explain the chart in one sentence, it’s probably doing too much.

Exercises

  • Create 3 charts that answer 3 different questions.
  • Write one insight and one caveat per chart.
  • List 5 follow-up questions your EDA suggests.
